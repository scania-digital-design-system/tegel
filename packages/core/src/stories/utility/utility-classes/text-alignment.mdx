# Text Alignment Utility Classes

The text-alignment utility classes provide a set of CSS classes to control the text-alignment property of elements.

## Available Classes

```css
.tds-u-textalign-start {
  text-align: start !important;
}

.tds-u-textalign-center {
  text-align: center !important;
}

.tds-u-textalign-end {
  text-align: end !important;
}
```

## Usage Examples

### Basic Usage

```html
<div class="tds-u-textalign-start">
  <!-- Content -->
</div>
```
