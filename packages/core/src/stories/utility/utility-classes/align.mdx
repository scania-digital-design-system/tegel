# Align Utility Classes

The align utility classes provide a set of CSS classes to control the align property of elements.

## Available Classes

```css
.tds-u-align-start {
  align-items: start !important;
}

.tds-u-align-center {
  align-items: center !important;
}

.tds-u-align-end {
  align-items: end !important;
}
```

## Usage Examples

### Basic Usage

```html
<div class="tds-u-align-start">
  <!-- Content -->
</div>
```
