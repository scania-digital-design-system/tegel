# Width Utility Classes

The width utility classes provide a set of CSS classes to control the width property of elements.

## Available Classes

```css
.tds-u-w-auto {
  width: auto !important;
}

.tds-u-w-100 {
  width: 100% !important;
}

.tds-u-w-75 {
  width: 75% !important;
}

.tds-u-w-50 {
  width: 50% !important;
}

.tds-u-w-25 {
  width: 25% !important;
}
```

## Usage Examples

### Basic Usage

```html
<div class="tds-u-w-auto">
  <!-- Content -->
</div>
```
