# Gap Utility Classes

The gap utility classes provide a set of CSS classes to control the gap property of elements.

## Available Classes

```css
.tds-u-gap1 {
  gap: 8px !important;
}

.tds-u-gap2 {
  gap: 16px !important;
}

.tds-u-gap3 {
  gap: 24px !important;
}

.tds-u-gap4 {
  gap: 32px !important;
}
```

## Usage Examples

### Basic Usage

```html
<div class="tds-u-gap1">
  <!-- Content -->
</div>
```
