# Position Utility Classes

The position utility classes provide a set of CSS classes to control the position property of elements.

## Available Classes

```css
.tds-u-sticky {
  position: sticky !important;
}

.tds-u-static {
  position: static !important;
}

.tds-u-relative {
  position: relative !important;
}

.tds-u-absolute {
  position: absolute !important;
}

.tds-u-fixed {
  position: fixed !important;
}
```

## Usage Examples

### Basic Usage

```html
<div class="tds-u-sticky">
  <!-- Content -->
</div>
```
