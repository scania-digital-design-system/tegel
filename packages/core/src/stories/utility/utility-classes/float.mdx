# Float Utility Classes

The float utility classes provide a set of CSS classes to control the float property of elements.

## Available Classes

```css
.tds-u-float-right {
  float: right !important;
}

.tds-u-float-left {
  float: left !important;
}

.tds-u-float-none {
  float: none !important;
}
```

## Usage Examples

### Basic Usage

```html
<div class="tds-u-float-right">
  <!-- Content -->
</div>
```
