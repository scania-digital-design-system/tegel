# Overflow Utility Classes

The overflow utility classes provide a set of CSS classes to control the overflow property of elements.

## Available Classes

```css
.tds-u-overflow-auto {
  overflow: auto !important;
}

.tds-u-overflow-scroll {
  overflow: scroll !important;
}

.tds-u-overflow-hidden {
  overflow: hidden !important;
}

.tds-u-overflow-visible {
  overflow: visible !important;
}
```

## Usage Examples

### Basic Usage

```html
<div class="tds-u-overflow-auto">
  <!-- Content -->
</div>
```
