# Opacity Utility Classes

The opacity utility classes provide a set of CSS classes to control the opacity property of elements.

## Available Classes

```css
.tds-u-opacity-100 {
  opacity: 1 !important;
}

.tds-u-opacity-75 {
  opacity: 0.75 !important;
}

.tds-u-opacity-50 {
  opacity: 0.5 !important;
}

.tds-u-opacity-25 {
  opacity: 0.25 !important;
}
```

## Usage Examples

### Basic Usage

```html
<div class="tds-u-opacity-100">
  <!-- Content -->
</div>
```
