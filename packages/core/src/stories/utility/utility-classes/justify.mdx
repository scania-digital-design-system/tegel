# Justify Utility Classes

The justify utility classes provide a set of CSS classes to control the justify property of elements.

## Available Classes

```css
.tds-u-justify-between {
  justify-content: space-between !important;
}

.tds-u-justify-around {
  justify-content: space-around !important;
}

.tds-u-justify-center {
  justify-content: center !important;
}
```

## Usage Examples

### Basic Usage

```html
<div class="tds-u-justify-between">
  <!-- Content -->
</div>
```
