# Visibility Utility Classes

The visibility utility classes provide a set of CSS classes to control the visibility property of elements.

## Available Classes

```css
.tds-u-visibility-hidden {
  visibility: hidden !important;
}

.tds-u-visibility-visible {
  visibility: visible !important;
}
```

## Usage Examples

### Basic Usage

```html
<div class="tds-u-visibility-hidden">
  <!-- Content -->
</div>
```
