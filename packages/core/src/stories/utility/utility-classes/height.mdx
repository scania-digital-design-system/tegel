# Height Utility Classes

The height utility classes provide a set of CSS classes to control the height property of elements.

## Available Classes

```css
.tds-u-h-auto {
  height: auto !important;
}

.tds-u-h-100 {
  height: 100% !important;
}

.tds-u-h-75 {
  height: 75% !important;
}

.tds-u-h-50 {
  height: 50% !important;
}

.tds-u-h-25 {
  height: 25% !important;
}
```

## Usage Examples

### Basic Usage

```html
<div class="tds-u-h-auto">
  <!-- Content -->
</div>
```
