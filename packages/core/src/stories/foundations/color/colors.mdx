## Usage

To use these colors in your components, you can:

1. Use CSS variables directly:

```css
.my-element {
  color: var(--tds-blue-500);
  background-color: var(--tds-grey-100);
}
```

2. Use utility classes:

```html
<p class="tds-text-blue-500">Blue text</p>
<div class="tds-background-grey-100">Grey background</div>
```

For more examples of how to use these colors, see the [Text Color](/docs/utilities-text-color--docs) and [Background Color](/docs/utilities-background-color--docs) utility documentation.
