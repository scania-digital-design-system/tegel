## Usage

To use these colors in your components, you can use CSS variables:

```css
.my-element {
  /* Using color scales */
  color: var(--tds-blue-500);
  background-color: var(--tds-grey-100);

  /* Using semantic colors */
  color: var(--tds-negative);
  background-color: var(--tds-information);
}
```

Note: The brand palette and some color scales are under review and may be changed or removed in future updates.
