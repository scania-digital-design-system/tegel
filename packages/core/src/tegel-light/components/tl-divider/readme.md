# tl-divider

The Divider component creates a visual separator between content sections.

## Usage

```html
<div class="tl-divider tl-divider--horizontal"></div>
```

<br />

### Required Stylesheets

```
@scania/tegel-lite/global.css
@scania/tegel-lite/tl-divider.css
```

## Elements

| Element       | HTML Element | Description             |
| ------------- | ------------ | ----------------------- |
| `.tl-divider` | `<div>`      | Main divider element    |

## Modifiers

### Divider Modifiers

Apply these classes to the `.tl-divider` element.

| Modifier                  | Description                     |
| ------------------------- | ------------------------------- |
| `.tl-divider--horizontal` | Horizontal divider (full width) |
| `.tl-divider--vertical`   | Vertical divider (full height)  |
